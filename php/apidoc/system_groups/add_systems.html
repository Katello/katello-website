POST /api/organizations/:organization_id/system_groups/:id/add_systems
Add systems to the group

Examples

POST /api/organizations/org_227e003/system_groups/1/add_systems
{
  "system_group": {
    "system_ids": [
      "3050eb45-0ec4-43ea-8913-83635f06a2b7"
    ]
  }
}
200
[
  {
    "name": "katello.example.com_227e003",
    "id": "3050eb45-0ec4-43ea-8913-83635f06a2b7"
  }
]

Params

Param name Description
organization_id
required

organization identifier


Value: string from 2 to 128 characters containting only alphanumeric characters, space, '_', '-' with no leading or trailing space..
id
required

Id of the system group


Value: string from 2 to 128 characters containting only alphanumeric characters, space, '_', '-' with no leading or trailing space..
system_group
required
Value: Hash
system_group[system_ids]
optional

Array of system ids


Value: Array
