POST /api/organizations/:organization_id/system_groups
Create a system group

Examples

POST /api/organizations/org_227e003/system_groups/
{
  "system_group": {
    "name": "system_group_227e003",
    "max_systems": "-1",
    "description": "group description"
  }
}
200
{
  "organization_id": 2,
  "name": "system_group_227e003",
  "max_systems": -1,
  "created_at": "2012-08-10T13:48:21Z",
  "description": "group description",
  "updated_at": "2012-08-10T13:48:21Z",
  "id": 1,
  "pulp_id": "org_227e003-system_group_227e003-9b0aed1f"
}

Params

Param name Description
organization_id
required

organization identifier


Value: string from 2 to 128 characters containting only alphanumeric characters, space, '_', '-' with no leading or trailing space..
system_group
required
Value: Hash
system_group[name]
required

System group name


Value: String
system_group[description]
optional
Value: String
system_group[max_systems]
required

Maximum number of systems in the group


Value: Integer
