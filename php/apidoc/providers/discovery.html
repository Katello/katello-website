POST /api/providers/:id/discovery
Discover repository urls with metadata and find candidate repos. Supports http, https and file based urls. Async task, returns the delayed job.

Examples

POST /api/providers/6/discovery
{
  "url": "http://lzap.fedorapeople.org/fakerepos/zoo5/"
}
