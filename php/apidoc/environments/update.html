PUT /api/environments/:id
Update an environment

PUT /api/organizations/:organization_id/environments/:id
Update an environment in an organization

Also see environments#create.

Examples

PUT /api/organizations/org_227e003/environments/5
{
  "environment": {
    "description": "Test environment",
    "name": "tstenv_227e003"
  }
}
200
{
  "created_at": "2012-08-10T13:45:23Z",
  "prior": "Library",
  "library": false,
  "organization_id": 2,
  "prior_id": 2,
  "id": 5,
  "updated_at": "2012-08-10T13:45:24Z",
  "organization": "org_227e003",
  "description": "Test environment",
  "name": "tstenv_227e003"
}
