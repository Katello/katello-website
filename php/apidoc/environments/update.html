PUT /api/environments/:id
Update an environment

PUT /api/organizations/:organization_id/environments/:id
Update an environment in an organization

Examples

PUT /api/organizations/org_q5ktCr6jM1oN/environments/9
{
  "environment": {
    "name": "tstenv_q5ktCr6jM1oN",
    "description": "Test environment"
  }
}
