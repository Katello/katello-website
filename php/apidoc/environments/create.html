POST /api/organizations/:organization_id/environments
Create an environment in an organization

Examples

POST /api/organizations/org_q5ktCr6jM1oN/environments/
{
  "environment": {
    "name": "env q5ktCr6jM1oN",
    "prior": 6
  }
}
200
{
  "name": "env q5ktCr6jM1oN",
  "id": 7,
  "updated_at": "2013-03-20T19:40:08Z",
  "prior_id": 6,
  "label": "env_q5ktCr6jM1oN",
  "organization": "org_q5ktCr6jM1oN",
  "organization_id": 4,
  "created_at": "2013-03-20T19:40:08Z",
  "prior": "Library",
  "library": false,
  "description": null
}

Params

Param name Description
organization_id
optional

organization identifier


Value: string from 2 to 128 characters containting only alphanumeric characters, space, '_', '-' with no leading or trailing space..
environment
required
Value: Hash
environment[name]
required

name of the environment (identifier)


Value: string from 2 to 128 characters containting only alphanumeric characters, space, '_', '-' with no leading or trailing space..
environment[description]
optional , nil allowed
Value: String
environment[prior]
required

identifier of an environment that is prior the new environment in the chain, it has to be either library or an environment at the end of the chain


Value: string from 2 to 128 characters containting only alphanumeric characters, space, '_', '-' with no leading or trailing space..
