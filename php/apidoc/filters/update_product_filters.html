PUT /api/organizations/:organization_id/products/:product_id/filters
Update package filters for product

Examples

PUT /api/organizations/org_227e003/products/1344606167242/filters
{
  "filters": [
    "filter_227e003"
  ]
}
200
[
  {
    "name": "filter_227e003",
    "id": 1,
    "package_list": [
      "package1",
      "package2"
    ],
    "description": "description",
    "created_at": "2012-08-10T13:42:58Z",
    "updated_at": "2012-08-10T13:42:58Z",
    "organization_id": 2,
    "pulp_id": "org_227e003-filter_227e003-940caf37"
  }
]

Params

Param name Description
organization_id
required

oranization identifier


Value: string from 2 to 128 characters containting only alphanumeric characters, space, '_', '-' with no leading or trailing space..
product_id
required

product identifier


Value: string from 2 to 128 characters containting only alphanumeric characters, space, '_', '-' with no leading or trailing space..
filters
required

Updated list of filters


Value: Array
