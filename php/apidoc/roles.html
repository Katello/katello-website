GET /api/roles

GET /api/users/:user_id/roles
List roles assigned to a user

Examples

GET /api/roles/?name=tmp_permission_role_q5ktCr6jM1oN
200

    "name": "tmp_permission_role_q5ktCr6jM1oN",
    "id": 8,
    "updated_at": "2013-03-20T19:43:19Z",
    "created_at": "2013-03-20T19:43:19Z",
    "locked": false,
    "description": null
  }
]

POST /api/roles
Create a role

POST /api/users/:user_id/roles
Create a role

Examples

POST /api/roles/
{
  "role": {
    "name": "tmp_permission_role_q5ktCr6jM1oN",
    "description": null
  }
}
200
{
  "name": "tmp_permission_role_q5ktCr6jM1oN",
  "id": 8,
  "updated_at": "2013-03-20T19:43:19Z",
  "created_at": "2013-03-20T19:43:19Z",
  "locked": false,
  "description": null
}

PUT /api/roles/:id
Update a role

PUT /api/users/:user_id/roles/:id
Update a role

Examples

PUT /api/roles/9
{
  "role": {
    "name": "role_",
    "description": "new description of the role"
  }
}
200
{
  "name": "role_",
  "id": 9,
  "updated_at": "2013-03-20T19:47:09Z",
  "created_at": "2013-03-20T19:47:07Z",
  "locked": false,
  "description": "new description of the role"
}

Params

Param name Description
role
required
Value: Hash
role[name]
optional
Value: String
role[description]
optional , nil allowed
Value: String
