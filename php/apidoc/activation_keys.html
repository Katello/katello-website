POST /api/activation_keys
Create an activation key

POST /api/environments/:environment_id/activation_keys
Create an activation key

Examples

POST /api/environments/3/activation_keys/
{
  "activation_key": {
    "usage_limit": -1,
    "description": "key description",
    "name": "activation_key_227e003"
  }
}
200
{
  "usage_limit": -1,
  "created_at": "2012-08-10T13:43:00Z",
  "environment_id": 3,
  "pools": [],
  "organization_id": 2,
  "id": 1,
  "updated_at": "2012-08-10T13:43:00Z",
  "system_template_id": null,
  "usage_count": 0,
  "description": "key description",
  "name": "activation_key_227e003",
  "user_id": 1
}

Params

Param name Description
activation_key
optional
Value: Hash
activation_key[name]
optional

activation key identifier (alphanum characters, space, - and _)


Value: string from 2 to 128 characters containting only alphanumeric characters, space, '_', '-' with no leading or trailing space..
activation_key[description]
optional , nil allowed
