PUT /api/roles/:id
Update a role

PUT /api/users/:user_id/roles/:id
Update a role

Examples

PUT /api/roles/7
{
  "role": {
    "name": "role_",
    "description": "new description of the role"
  }
}
200
{
  "name": "role_",
  "id": 7,
  "description": "new description of the role",
  "created_at": "2012-08-10T13:58:00Z",
  "updated_at": "2012-08-10T13:58:02Z",
  "locked": false
}
