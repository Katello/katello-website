System group errata

methods for handling erratas on system group level

>>>

GET /api/organizations/:organization_id/system_groups/:system_group_id/errata
Get list of errata associated with the group

Params

Param name Description
organization_id
required

oranization identifier


Value: string from 2 to 128 characters containting only alphanumeric characters, space, '_', '-' with no leading or trailing space..
system_group_id
required

system_group identifier


Value: string from 2 to 128 characters containting only alphanumeric characters, space, '_', '-' with no leading or trailing space..
type
optional

Filter errata by type


Value: Must be one of: bugfix, enhancement, security.
