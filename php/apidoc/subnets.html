Subnets

The Subnet API is available only if support for Foreman is installed.

>>>

GET /api/subnets
Get list of subnets available in Foreman

>>>

GET /api/subnets/:id
Show an subnet

Params

Param name Description
id
optional

subnet name


Value: String
>>>

POST /api/subnet
Create new subnet in Foreman

subnet
required

subnet info

required

subnet name


Value: String
