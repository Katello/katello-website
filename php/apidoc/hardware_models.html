Hardware models

The HardwareModels API is available only if support for Foreman is installed.

>>>

GET /api/hardware_models
Get list of hardware models available in Foreman

>>>

GET /api/hardware_models/:id
Show an hardware model

Params

Param name Description
id
optional

hardware model name


Value: String
>>>

POST /api/hardware_models
Create new hardware model in Foreman

Value: String
