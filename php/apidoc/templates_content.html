Templates content

Provides interface to the template content.

>>>

POST /api/templates/:template_id/packages
Add a package

Examples

POST /api/templates/1/packages/
{
  "name": "cheetah"
}
200
Added package 'cheetah'

Params

Param name Description
template_id
required

template numeric identifier


Value: number.
name
required

package identifier


Value: string from 2 to 128 characters containting only alphanumeric characters, space, '_', '-' with no leading or trailing space..
>>>
