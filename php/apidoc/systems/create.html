Params

Param name Description
facts
optional , nil allowed

Key-value hash of system-specific facts


Value: Hash
installedProducts
optional , nil allowed

List of products installed on the system


Value: Array
name
required

Name of the system


Value: String
type
required

Type of the system, it should always be ‘system’


Value: String
serviceLevel
optional , nil allowed

A service level for auto-healing process, e.g. SELF-SUPPORT


Value: String
location
optional

Physical of the system


Value: String
content_view_id
optional
Value: string from 2 to 128 characters containting only alphanumeric characters, space, '_', '-' with no leading or trailing space..
environment_id
optional
Value: string from 2 to 128 characters containting only alphanumeric characters, space, '_', '-' with no leading or trailing space..
