PUT /api/gpg_keys/:id
Update a gpg key

Examples

PUT /api/gpg_keys/2/
{
  "gpg_key": {
    "name": "Renamed-Dummy-Package-Generator"
  }
}
200
{
  "name": "Renamed-Dummy-Package-Generator",
  "id": 2,
  "updated_at": "2013-03-20T19:42:42Z",
  "content": "gpg create from input                                                                                                                                                   [ \u001b[32mOK\u001b[m\u000f ]\n",
  "organization_id": 4,
  "created_at": "2013-03-20T19:42:40Z"
}

Params

Param name Description
gpg_key
required
Value: Hash
gpg_key[name]
optional

identifier of the gpg key


Value: string from 2 to 128 characters containting only alphanumeric characters, space, '_', '-' with no leading or trailing space..
gpg_key[content]
optional

public key block in DER encoding


Value: String
