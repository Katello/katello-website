POST /api/changesets/:changeset_id/errata
Add an errata to a changeset

Examples

POST /api/changesets/2/errata/
{
  "product_id": "1344606167242",
  "erratum_id": "RHEA-2010:0001"
}
200
Added erratum 'RHEA-2010:0001'

Params

Param name Description
erratum_id
optional

The id of the errata to add


Value: number.
product_id
optional

The product which contains the errata


Value: number.
