POST /api/changesets/:changeset_id/repositories
Add a repository to a changset

Examples

POST /api/changesets/2/repositories/
{
  "repository_id": 2,
  "product_id": "1344606167242"
}
200
Added repository 'repo_227e003'

Params

Param name Description
repository_id
optional

The id of the repository to add


Value: number.
