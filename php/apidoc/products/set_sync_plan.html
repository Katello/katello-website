POST /api/organizations/:organization_id/products/:id/sync_plan
Assign sync plan to product

Examples

POST /api/organizations/org_q5ktCr6jM1oN/products/1363808411864/sync_plan
{
  "plan_id": 1
}
200
Synchronization plan assigned.

Params

Param name Description
organization_id
optional

organization identifier


Value: string from 2 to 128 characters containting only alphanumeric characters, space, '_', '-' with no leading or trailing space..
id
optional

product numeric identifier


Value: number.
plan_id
optional

Plan numeric identifier


Value: number.
