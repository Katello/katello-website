Sync plans

Synchronization plans are used to configure the scheduled synchronization of the repository with the upstream.

>>>

GET /api/organizations/:organization_id/sync_plans
List sync plans

Examples

GET /api/organizations/org_q5ktCr6jM1oN/sync_plans/
200
[
  {
    "name": "plan_2",
    "id": 2,
    "updated_at": "2013-03-20T19:46:06Z",
    "interval": "none",
    "organization_id": 4,
    "created_at": "2013-03-20T19:46:06Z",
    "description": "plan 2 desc.",
    "sync_date": "2013-03-21T13:00:00Z"
  },
  {
    "name": "plan_1",
    "id": 1,
    "updated_at": "2013-03-20T19:46:06Z",
    "interval": "weekly",
    "organization_id": 4,
    "created_at": "2013-03-20T19:46:06Z",
    "description": "plan 1 desc.",
    "sync_date": "2013-03-21T13:00:00Z"
  }
]

Params

Param name Description
organization_id
required

oranization identifier


Value: string from 2 to 128 characters containting only alphanumeric characters, space, '_', '-' with no leading or trailing space..
name
optional

filter by name


Value: String
sync_date
optional

filter by sync date


Value: String
interval
optional

filter by interval


Value: Must be one of: none, hourly, daily, weekly.

>>>

POST /api/organizations/:organization_id/sync_plans
Create a sync plan

Examples

POST /api/organizations/org_q5ktCr6jM1oN/sync_plans/
{
  "sync_plan": {
    "name": "plan_1",
    "interval": "weekly",
    "description": "plan 1 desc.",
    "sync_date": "2013-03-21T09:00:00-04:00"
  }
}
200
{
  "name": "plan_1",
  "id": 1,
  "updated_at": "2013-03-20T19:46:06Z",
  "interval": "weekly",
  "organization_id": 4,
  "created_at": "2013-03-20T19:46:06Z",
  "description": "plan 1 desc.",
  "sync_date": "2013-03-21T13:00:00Z"
}

Params

Param name Description
organization_id
required

oranization identifier


Value: string from 2 to 128 characters containting only alphanumeric characters, space, '_', '-' with no leading or trailing space..
sync_plan
required
Value: Hash
sync_plan[name]
required

sync plan name


Value: String
sync_plan[interval]
optional , nil allowed

how often synchronization should run


Value: Must be one of: none, hourly, daily, weekly.
sync_plan[sync_date]
optional , nil allowed

start datetime of synchronization


Value: String
sync_plan[description]
optional , nil allowed

sync plan description


Value: String
