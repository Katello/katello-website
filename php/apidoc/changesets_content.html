POST /api/changesets/:changeset_id/products
Add a product to a changeset

Examples

POST /api/changesets/1/products/
{
  "product_id": "1344606167242"
}
200
Added product 'fewups_product_227e003'

Params

Param name Description
product_id
optional

The id of the product which should be added


Value: number.
>>>

DELETE /api/changesets/:changeset_id/products/:id
Removes a product from a changeset

Examples

DELETE /api/changesets/1/products/1344606167242/
{
  "content_id": "1344606167242"
}
200
Removed product '1344606167242'

Params

Param name Description
content_id
optional

The id of the product to remove


Value: number.
