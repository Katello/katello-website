System group packages

methods for handling packages on system group level

>>>

POST /api/organizations/:organization_id/system_groups/:system_group_id/packages
Install packages remotely

Examples

POST /api/organizations/org_227e003/system_groups/1/packages
{
  "packages": [
    "cheetah"
  ]
}
202
{
  "parameters": {
    "packages": [
      "cheetah"
    ]
  },
  "status_message": "Installing Package...",
  "id": 1,
  "state": "running",
  "task_type": "package_install",
  "finish_time": null,
  "tasks": [
    {
      "result": {
        "errors": [
          null,
          null
        ]
      },
      "id": 15,
      "start_time": null,
      "uuid": "0e0f181c-e2f2-11e1-85ad-525400b68f90",
      "state": "waiting",
      "finish_time": null,
      "progress": null
    }
  ],
  "created_at": "2012-08-10T13:48:24Z",
  "pulp_id": "0384935e-4216-4d38-9832-6be39e379aba"
}

Params

Param name Description
organization_id
required

oranization identifier


Value: string from 2 to 128 characters containting only alphanumeric characters, space, '_', '-' with no leading or trailing space..
system_group_id
required

system_group identifier


Value: string from 2 to 128 characters containting only alphanumeric characters, space, '_', '-' with no leading or trailing space..
packages
optional

List of package names to install


Value: Array
groups
optional

List of package group names to install


Value: Array
