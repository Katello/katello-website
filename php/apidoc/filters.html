Filters

methods for setting package filters for repository, product or organization

>>>

GET /api/organizations/:organization_id/filters
List package filters for an organization

Examples

GET /api/organizations/org_227e003/filters
200
[
  {
    "name": "filter_227e003",
    "id": 1,
    "package_list": [
      "package1",
      "package2"
    ],
    "description": "description",
    "created_at": "2012-08-10T13:42:58Z",
    "updated_at": "2012-08-10T13:42:58Z",
    "organization_id": 2,
    "pulp_id": "org_227e003-filter_227e003-940caf37"
  }
]

Params

Param name Description
organization_id
required

oranization identifier


Value: string from 2 to 128 characters containting only alphanumeric characters, space, '_', '-' with no leading or trailing space..
>>>

POST /api/organizations/:organization_id/filters
Create package filter for an organization

Examples

POST /api/organizations/org_227e003/filters
{
  "name": "filter_227e003",
  "package_list": [
    "package1",
    "package2"
  ],
  "description": "description"
}

  "id": 1,

}

Param name Description

required

PUT /api/organizations/:organization_id/filters/:id
Update package filter for an organization

Examples

PUT /api/organizations/org_227e003/filters/filter2_227e003
{
  "packages": [
    "package3",
    "package4",
    "package5"
  ]
}
200
{
  "name": "filter2_227e003",
  "id": 2,
  "package_list": [
    "package3",
    "package4",
    "package5"
  ],
  "description": "description",
  "created_at": "2012-08-10T13:45:49Z",
  "updated_at": "2012-08-10T13:45:49Z",
  "organization_id": 2,
  "pulp_id": "org_227e003-filter2_227e003-814619c9"
}

Params

Param name Description
organization_id
required

oranization identifier


Value: string from 2 to 128 characters containting only alphanumeric characters, space, '_', '-' with no leading or trailing space..
id
required

filter identifier


Value: string from 2 to 128 characters containting only alphanumeric characters, space, '_', '-' with no leading or trailing space..
name
optional

filter's new name


Value: String
packages
required

Updated list of the packages to filter


Value: Array
