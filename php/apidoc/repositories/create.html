Params

Param name Description
name
required
Value: String
organization_id
required

id of an organization the repository will be contained in


Value: string from 2 to 128 characters containting only alphanumeric characters, space, '_', '-' with no leading or trailing space..
product_id
required

id of a product the repository will be contained in


Value: number.
url
required

repository source url
gpg_key_name
optional

name of a gpg key that will be assigned to the new repository


Value: String
