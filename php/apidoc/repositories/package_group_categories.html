GET /api/repositories/:id/package_group_categories
List all package group categories in a repository

Examples

GET /api/repositories/2/package_group_categories/
200
[
  {
    "repo_defined": true,
    "immutable": true,
    "name": "all",
    "_id": "all",
    "display_order": 99,
    "description": null,
    "id": "all",
    "translated_description": {},
    "translated_name": {},
    "packagegroupids": [
      "mammal",
      "bird"
    ]
  }
]

Params

Param name Description
id
required
Value: string from 2 to 128 characters containting only alphanumeric characters, space, '_', '-' with no leading or trailing space..
