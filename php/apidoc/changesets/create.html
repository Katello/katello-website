POST /api/organizations/:organization_id/environments/:environment_id/changesets
Create a changeset

Examples

POST /api/organizations/org_q5ktCr6jM1oN/environments/7/changesets/
{
  "changeset": {
    "name": "changeset_q5ktCr6jM1oN",
    "type": "PROMOTION",
    "description": null
  }
}
200
{
  "name": "changeset_q5ktCr6jM1oN",
  "id": 1,
  "action_type": "promotion",
  "updated_at": "2013-03-20T19:40:31Z",
  "task_status_id": null,
  "created_at": "2013-03-20T19:40:31Z",
  "state": "new",
  "promotion_date": null,
  "description": null,
  "environment_id": 7
}

The description of the changeset

required
