PUT /api/organizations/:organization_id/sync_plans/:id
Update a sync plan

Examples

PUT /api/organizations/org_227e003/sync_plans/1
{
  "sync_plan": {
    "sync_date": "2012-08-11T10:10:10+02:00",
    "name": "plan_1_updated",
    "interval": "daily",
    "description": "new plan 1 desc."
  }
}
200
{
  "sync_date": "2012-08-11T08:10:10Z",
  "name": "plan_1_updated",
  "id": 1,
  "interval": "daily",
  "description": "new plan 1 desc.",
  "created_at": "2012-08-10T13:56:21Z",
  "updated_at": "2012-08-10T13:56:22Z",
  "organization_id": 2
}

Params

Param name Description

required

oranization identifier


Value: string from 2 to 128 characters containting only alphanumeric characters, space, '_', '-' with no leading or trailing space..
id
required

sync plan numeric identifier


Value: number.

required
Value: Hash

optional

sync plan description
sync_plan[interval]
optional

how often synchronization should run


Value: Must be one of: none, hourly, daily, weekly.
