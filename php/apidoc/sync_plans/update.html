PUT /api/organizations/:organization_id/sync_plans/:id
Update a sync plan

Examples

PUT /api/organizations/org_q5ktCr6jM1oN/sync_plans/1
{
  "sync_plan": {
    "name": "plan_1_updated",
    "interval": "daily",
    "description": "new plan 1 desc.",
    "sync_date": "2013-03-21T10:10:10-04:00"
  }
}
200
{
  "name": "plan_1_updated",
  "id": 1,
  "updated_at": "2013-03-20T19:46:07Z",
  "interval": "daily",
  "organization_id": 4,
  "created_at": "2013-03-20T19:46:06Z",
  "description": "new plan 1 desc.",
  "sync_date": "2013-03-21T14:10:10Z"
}

Params

Param name Description
organization_id
required

oranization identifier


Value: string from 2 to 128 characters containting only alphanumeric characters, space, '_', '-' with no leading or trailing space..
id
required

sync plan numeric identifier


Value: number.
sync_plan
required
Value: Hash
sync_plan[name]
optional

sync plan name


Value: String
sync_plan[interval]
optional , nil allowed

how often synchronization should run


Value: Must be one of: none, hourly, daily, weekly.
sync_plan[sync_date]
optional , nil allowed

start datetime of synchronization


Value: String
sync_plan[description]
optional , nil allowed

sync plan description


Value: String
