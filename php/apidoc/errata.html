Errata

>>>

GET /api/errata
List errata

GET /api/repositories/:repository_id/errata
List errata

Either :repoid or :product_id is required.

Errors

401 Unauthorized
404 Not found
400 Repo id or environment must be provided

Examples

GET /api/errata?environment_id=6
200

Params

Param name Description
environment_id
optional

Id of environment containing the errata.


Value: number.
product_id
optional

The product which contains the errata.


Value: number.
repoid
optional

Id of repository containing the errata.


Value: number.
severity
optional

Severity of errata. Usually one of: Critical, Important, Moderate, Low. Case insensitive.


Value: String
type
optional

Type of errata. Usually one of: security, bugfix, enhancement. Case insensitive.


Value: String
